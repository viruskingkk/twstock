# 台股六脈神劍選股系統

## 版本說明

目前版本：**台灣50預設_多空分類篩選版**

這是一個本機 HTML 台股技術面選股工具，整合：

- 六脈神劍
- 求敗九劍
- 十八漲
- 多空勝率 / 方向機率
- 多空分類篩選
- 持股追蹤
- 賣出 / 贖回提示
- 外部網站資料連結
- CSV 離線資料匯入
- Python 批次下載歷史資料

> 本工具僅作為技術指標篩選與持股追蹤輔助，不構成投資建議。

## 快速開始

### 方式一：直接用台灣50預設股票池

1. 解壓縮 ZIP。
2. 用 Chrome 或 Edge 開啟 `台股六脈神劍選股系統.html`。
3. 「① 股票池與資料來源」預設已填入台灣50成分股。
4. 按下「開始抓取並選股」。
5. 等待分析完成後查看：
   - 六脈神劍
   - 求敗九劍
   - 十八漲
   - 多空勝率
   - 持股追蹤

適合少量股票、台灣50、手動代號清單測試。

---

### 方式二：用 Python 下載 CSV 後離線分析

瀏覽器直接抓取官方資料時，可能會遇到 CORS 或限流問題。建議全市場分析時改用 Python 先下載 CSV。

#### 1. 安裝 Python

請安裝 Python 3.10 以上版本。

#### 2. 下載普通股全市場資料

直接雙擊：

```text
下載普通股全市場.bat
```

或手動執行：

```bash
python tw_stock_downloader.py --all --out history.csv --months 4 --sleep 0.8 --resume
```

參數說明：

```text
--all       下載上市 + 上櫃普通股
--out       輸出 CSV 檔案名稱
--months    下載最近幾個月資料
--sleep     每檔下載間隔秒數，降低被限流機率
--resume    續跑模式，已下載過的股票會自動跳過
```

#### 3. 在 HTML 匯入 CSV

1. 開啟 `台股六脈神劍選股系統.html`。
2. 找到「上傳離線下載的 CSV 資料」。
3. 選擇 `history.csv`。
4. 按「用此 CSV 選股」。

如果瀏覽器讀取 CSV 失敗，可用記事本打開 `history.csv`，全選複製後貼到 CSV 備援文字區。

## CSV 格式

系統支援下列欄位：

```csv
code,name,market,date,open,high,low,close,volume
2330,台積電,twse,2026-06-28,1000,1010,995,1005,25000000
```

欄位說明：

| 欄位 | 說明 |
|---|---|
| code | 股票代號 |
| name | 股票名稱 |
| market | 市場，`twse` 或 `tpex` |
| date | 日期 |
| open | 開盤價 |
| high | 最高價 |
| low | 最低價 |
| close | 收盤價 |
| volume | 成交量，股數 |

## 股票池設定

### 預設股票池

目前 HTML 預設為台灣50成分股，適合快速測試與大型權值股篩選。

### 手動輸入股票池

可在股票池文字框中輸入股票代號，例如：

```text
2330,2317,2454,2308,2382
```

支援逗號、空白、換行分隔。

### 全市場分析建議

全市場建議不要直接用瀏覽器即時抓行情，請使用 Python 下載 CSV 後再匯入。

穩定參數建議：

```bash
python tw_stock_downloader.py --all --out history.csv --months 4 --sleep 0.8 --resume
```

如果 TPEx 上櫃資料大量失敗，可降低速度：

```bash
python tw_stock_downloader.py --all --out history.csv --months 4 --sleep 1.5 --resume
```

## 名冊篩選邏輯

新版 `--all` 預設只抓：

- 上市普通股
- 上櫃普通股

預設排除：

- ETF
- ETN
- 權證
- 債券型商品
- 受益證券
- 其他非普通股商品

如需包含 ETF，可使用：

```bash
python tw_stock_downloader.py --all --include-etf --out history.csv --months 4 --sleep 0.8 --resume
```

若看到股票數接近一萬檔，通常代表跑到舊版 downloader，或使用了過寬的商品名冊。

## 功能說明

## 1. 六脈神劍

分成多方與空方條件。

多方：

- 第一劍【翻多】
- 第三劍【轉強】
- 第五劍【收高】

空方：

- 第二劍【翻空】
- 第四劍【轉弱】
- 第六劍【收低】

每個規則區塊支援：

- 收合 / 展開
- 篩選本條件
- 全選 / 全不選
- 個股追蹤
- 外部網站連結

---

## 2. 求敗九劍

求敗九劍主要視為空方或轉弱條件，用於辨識風險、減碼、停損觀察。

---

## 3. 十八漲

十八漲前 12 條視為多方條件。

第 13～18 條為排行榜型規則，系統將其視為參考訊號，不直接灌入主要多空分數，避免小股票池時所有股票都被選中。

---

## 4. 多空勝率

多空勝率不是歷史回測保證勝率，而是根據目前技術規則命中情況推估的方向機率。

系統會顯示：

- 分析股票數
- 偏多以上
- 偏空以上
- 多空打架
- 強多
- 偏多
- 中性觀察
- 偏空
- 強空

統計卡片可直接點擊篩選。

### 分類說明

| 分類 | 意義 |
|---|---|
| 強多 | 多方條件明顯大於空方條件 |
| 偏多 | 多方條件較多，仍可觀察 |
| 中性觀察 | 多空訊號不明顯 |
| 偏空 | 空方條件較多，需注意風險 |
| 強空 | 空方條件明顯，偏向賣出或避開 |
| 多空打架 | 多方與空方同時明顯，波動或轉折風險較高 |

---

## 5. 持股追蹤

可輸入已買進股票，系統會根據最新分析結果顯示持股狀態。

格式：

```text
代號,買進價,股數,購買金額,備註
```

範例：

```text
3481,18.5,1000,18500,面板追蹤
2330,950,,95000,長線核心
```

若只填購買金額、不填股數，系統會自動估算：

```text
股數 = 購買金額 / 買進價
```

持股追蹤會顯示：

- 現價
- 買進價
- 股數
- 購買金額
- 目前市值
- 損益金額
- 損益百分比
- 多方機率
- 空方機率
- 賣出 / 贖回提示

---

## 6. 賣出 / 贖回提示

可設定：

- 空方機率達幾 % 時提醒賣出 / 贖回
- 停損跌幅幾 % 時提醒停損 / 贖回

提示類型包含：

- 續抱
- 續抱觀察
- 減碼觀察
- 賣出 / 贖回
- 停損 / 贖回

## 7. 外部網站資料

每檔股票可快速開啟外部資料：

- Yahoo 股市
- Goodinfo
- CMoney
- Fugle
- TradingView

Yahoo 範例：

```text
https://tw.stock.yahoo.com/quote/3481.TW
```

批次開啟外部網站時，Chrome 可能會擋彈出視窗。請允許本機 HTML 的彈出視窗。

## 常見問題

### Q1：為什麼直接用 HTML 抓全市場會失敗？

因為瀏覽器有 CORS 限制，TWSE / TPEx 部分 API 不允許本機 HTML 直接抓取。全市場建議使用 Python downloader 先下載 CSV。

---

### Q2：為什麼以前抓到 10989 檔？

那是名冊抓太寬，把 ETF、ETN、權證、債券、受益證券等非普通股商品也抓進來。新版 `--all` 已預設只抓普通股。

---

### Q3：為什麼 TPEx 上櫃有些股票查無資料？

可能是官方端點限流、資料延遲、端點格式變動，或該股票近期資料不足。downloader 會嘗試 TPEx 與 Yahoo 備援。

---

### Q4：CSV 選了但讀不到怎麼辦？

請改用備援方式：

1. 用記事本打開 `history.csv`。
2. 全選複製。
3. 貼到 HTML 裡的 CSV 備援文字區。
4. 按「用此 CSV 選股」。

---

### Q5：多空勝率可以當成真正勝率嗎？

不建議。這裡的多空勝率是根據技術規則命中數推估的方向機率，適合做排序、觀察、風險追蹤，不是歷史回測保證勝率。

## 注意事項

- 本工具不是投資建議。
- 技術指標需要足夠交易日資料，建議至少 30 個交易日以上。
- 全市場資料量大，建議使用 CSV 離線模式。
- 官方 API 可能限流，下載時建議保留 `--sleep 0.8` 或更高。
- 台灣50成分股會定期調整，預設清單日後可能需要更新。
