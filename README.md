# 台股六脈神劍選股系統｜藏鋒榜精篩版

本版整合：六脈神劍、求敗九劍、十八漲、十絕陣、多空勝率、藏鋒榜精篩、持股追蹤與賣出/贖回提示。

## 建議使用方式

1. 解壓縮 ZIP。
2. 雙擊 `start_local_server.bat` 或 `啟動本機網頁版.bat`。
3. 用瀏覽器開啟本機網址：

```text
http://127.0.0.1:8765/台股六脈神劍選股系統.html
```

4. 先用 downloader 下載資料，或直接在 HTML 匯入 `history.csv`。
5. 按「用CSV資料選股」。
6. 到「藏鋒榜」分頁做最後精篩。

> 建議不要直接雙擊 HTML 用 `file://` 開啟，瀏覽器可能會限制外部連結或檔案讀取。

## 下載普通股全市場資料

推薦執行：

```bat
download_common_stocks_with_capital.bat
```

或手動執行：

```bat
python tw_stock_downloader.py --all --out history.csv --months 4 --sleep 0.8 --resume
```

CSV 需要至少包含：

```csv
code,name,market,industry,capital,date,open,high,low,close,volume
```

其中 `capital` 用於計算週轉率。

## 藏鋒榜是什麼？

前面的規則會選出很多股票，藏鋒榜是最後一層「精選排序 / 汰弱留強」。

它會綜合：

- 多方機率下限
- 空方機率上限
- 共振分數下限
- 多方命中數下限
- 十絕陣多方命中數下限
- 空方命中數上限
- 是否排除多空打架
- 是否排除偏空 / 強空
- 是否必須有週轉率
- 只顯示前 N 名

## 篩選強度

內建 5 種模式：

| 模式 | 用途 |
|---|---|
| 寬鬆 | 想多看候選股 |
| 標準 | 一般選股觀察 |
| 嚴格 | 預設模式，適合全市場精篩 |
| 極嚴格 | 只留最強共振股 |
| 自訂 | 手動調整所有門檻 |

## 終極分數

藏鋒榜排序使用「終極分數」，大致邏輯為：

```text
終極分數 = 共振分數
         + 多方機率加權
         - 空方機率加權
         + 十絕陣多方加分
         - 十絕陣空方扣分
         + 週轉率加分
```

這個分數不是投資報酬保證，只是用來排序與降低觀察名單數量。

## 持股追蹤

可以在選股結果或藏鋒榜直接按「追蹤」，加入持股追蹤區。持股區支援：

```text
代號,買進價,股數,購買金額,備註
```

例如：

```text
2330,950,,95000,長線核心
3481,18.5,1000,18500,面板追蹤
```

若只填購買金額、不填股數，系統會用買進價換算股數。

## 注意事項

- 本工具是技術面篩選輔助，不構成投資建議。
- 多空勝率是規則命中推估，不是歷史回測勝率。
- 全市場資料量大，建議使用 CSV 離線分析，比瀏覽器直接抓官方 API 穩定。

## 2026-06-30 更新：藏鋒榜多空雙榜

藏鋒榜新增「榜單方向」：

- 多方精選：用多方機率、空方上限、共振分、十絕陣多方命中做精篩。
- 空方警示：用空方機率、多方上限、反向共振分、十絕陣空方命中做精篩。
- 多空並列：同時呈現多方候選與空方候選，結果會在「榜單方向」欄位標示多方或空方。

原本「排除偏空/強空」改成「排除反向強勢」：

- 多方精選時，會排除偏空/強空。
- 空方警示時，會排除偏多/強多。
- 多空並列時，依各方向條件判斷。

若用 file:// 直接開 HTML 仍看到瀏覽器安全警告，請改用 ZIP 內的 `start_local_server.bat`，透過 localhost 開啟。
