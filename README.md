# 台股六脈神劍選股系統｜十絕陣整合版

本版本整合原有「六脈神劍、求敗九劍、十八漲、多空勝率、持股追蹤、產業分類、離線股本週轉率」功能，並新增第四套技術模組：**十絕陣**。

> 本工具僅做技術面篩選、追蹤與風險提示，不構成投資建議。

## 新增：十絕陣

十絕陣補強原本 KD、MACD、均線與乖離率邏輯，新增 10 個可由 OHLCV 離線資料直接計算的技術條件。

### 多方五絕

| 招式 | 指標 | 條件 |
|---|---|---|
| 一絕｜青龍列陣 | 多頭排列 | 收盤價 > MA5 > MA10 > MA20 |
| 三絕｜回陽一線 | 均線上彎 | MA5 與 MA10 同步上彎 |
| 五絕｜雷動九霄 | 量價齊鳴 | 價漲，且成交量 > 5日均量 × 1.2 |
| 七絕｜破雲出岫 | 突破20日高點 | 收盤突破前20日最高收盤，且量能高於5日均量 |
| 九絕｜赤焰長虹 | 長紅K | 實體漲幅 > 2%，且收盤接近最高價 |

### 空方五絕

| 招式 | 指標 | 條件 |
|---|---|---|
| 二絕｜白虎伏殺 | 空頭排列 | 收盤價 < MA5 < MA10 < MA20 |
| 四絕｜墜陰一線 | 均線下彎 | MA5 與 MA10 同步下彎 |
| 六絕｜破軍下殺 | 價跌量增 | 價跌，且成交量 > 5日均量 × 1.2 |
| 八絕｜墜淵失守 | 跌破20日低點 | 收盤跌破前20日最低收盤 |
| 十絕｜黑煞斷魂 | 長黑K | 實體跌幅 > 2%，且收盤接近最低價 |

## 對多空勝率的影響

十絕陣會納入「多空勝率」計算：

- 多方五絕命中時，增加多方規則數。
- 空方五絕命中時，增加空方規則數。
- 突破、跌破、量價、長紅、長黑等會被視為今日觸發訊號，影響共振分。

## 建議使用流程

1. 先用 ZIP 內的 BAT 或 Python downloader 下載資料。
2. 產生 `history.csv`。
3. 打開 `台股六脈神劍選股系統.html`。
4. 在「② 上傳離線下載的CSV資料」匯入 `history.csv`。
5. 點「用CSV資料選股」。
6. 查看：
   - 六脈神劍
   - 求敗九劍
   - 十八漲
   - 十絕陣
   - 多空勝率
   - 持股追蹤

## 常用 BAT

- `download_common_stocks_with_capital.bat`：下載上市 + 上櫃普通股，含股本。
- `下載普通股全市場_含股本.bat`：中文檔名版本。
- `download_common_plus_etf.bat`：普通股加 ETF。

## CSV 格式

建議使用新版 downloader 產生的 `history.csv`：

```csv
code,name,market,industry,capital,date,open,high,low,close,volume
2330,台積電,twse,14,2593.04,2026-06-01,900,910,895,905,12345678
```

HTML 匯入後會自動：

- 讀取產業欄位，並將產業代碼轉為中文名稱。
- 讀取股本欄位，用於計算週轉率。
- 計算 KD、MACD、均線、乖離率、量價、突破/跌破、K線型態。

## 週轉率公式

```text
週轉率 = 成交量(張) ÷ 股本(億) ÷ 100
```

若 `history.csv` 中有 `capital` 欄位，系統會自動計算，不需另外上傳 `stock_meta.csv`。

## 備註

`stock_meta.csv` 是給你用 Excel 檢查基本資料的備查檔，不需要放到特定資料夾，也不需要在 HTML 額外上傳。
